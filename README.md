# OmegaUI Command Line Interface
>OmegaUI 快速构建工具

### 依赖
> node >= 9.0.0 && npm >= 6.0.0

### 安装
```bash
$ npm install omegaui-cli
```

### 使用
```bash
$ omegaui-cli init my-project
```
>可以使用简写 i 替代 init 。

### 开发环境
```bash
cd my-project && npm run dev
```

### 生产环境
```bash
cd my-project && npm run build
```
