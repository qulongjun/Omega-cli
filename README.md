# OmegaUI Command Line Interface
>OmegaUI 快速构建工具

### 依赖
> node >= 9.0.0
> npm >= 6.0.0

### 安装
```bash
$ npm install omegaui-cli
```

### 使用
```bash
$ omega-cli init my-project
```

### 开发环境
```bash
cd my-project && npm run dev
```

### 生产环境
```bash
cd my-project && npm run build
```

>稳定版本：功能均经过完整测试，状态稳定，推荐使用。
